Amazing how only after a month of 42 I'm confident enough to figure out how to host a web page with a custom domain and get it up in under and hour.

Here's how I did it:

  • bought a domain name (I used godaddy)
  • Setup github page on my github using this guide:
    • http://andrewsturges.com/blog/jekyll/tutorial/2014/!/06/github-and-godaddy.htm
  • set up the DNS records at godaddy using this guide:
    • http://andrewsturges.com/blog/jekyll/tutorial/2014/!/06/github-and-godaddy.html
  • Git cloned and then started playing with the index.html - that's how i added in this text!

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
